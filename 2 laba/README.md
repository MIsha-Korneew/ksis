# Лабораторная работа 2 — Traceroute (ICMP)

## Файлы

- `traceroute_icmp.py` — утилита трассировки маршрута (аналог `tracert`), raw-сокеты, ручная сборка ICMP.

## Запуск

Требуются права **администратора** (Windows).

```text
py traceroute_icmp.py <хост>
py traceroute_icmp.py google.com -d
```
